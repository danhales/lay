# Linear Algebra, Yeah!

A python library where I explore metaprogramming tricks through the lens of my old linear algebra textbook from college (written by David C. Lay).

This library will provide a from-scratch implementation of various linear algebra objects (Vector, Matrix, etc.).

This is intended to be educational, not a production library. For optimized vector operations, you'll probably want to use a library like numpy. If you're a mathematician and want to leverage your understanding of matrix operations to teach yourself python, this is the repo for you.
